# Developer Guide


## Installation

First install development tools:

* Qt Creator ([pacman](https://www.archlinux.org/packages/?name=qtcreator))

Clone the repo via git:

```bash
git clone --depth=1 https://github.com/EnthDev/dedicatedslave.git dedicatedslave
cd dedicatedslave
```
